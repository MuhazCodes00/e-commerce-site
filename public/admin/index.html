To enable read/write actions from the storefront browser, make sure your Cloud Firestore rules allow read and write operations. Paste this in your **Firebase Console ➔ Firestore Database ➔ Rules** tab:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }
